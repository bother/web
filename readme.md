# Bother

What bothers you?

## Development

### Prerequisites

1. Git
1. Node
1. Yarn

### Environment

1. Check `.env.example`
1. `cp .env.example .env`
1. Fill in the values in `.env`

### Running

1. Clone this repo
1. `yarn` to install dependencies
1. `yarn dev` to boot up Next server
